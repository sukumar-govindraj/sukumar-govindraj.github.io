# Sukumar Govindaraj Portfolio

Professional GitHub Pages portfolio for Sukumar Govindaraj.

## Positioning

Snowflake-certified data scientist building agentic AI systems for supply chain, audit, maintenance, and enterprise operations.

## Design direction

Inspired by modern dark developer-tool landing pages such as Hermes Agent, Linear, Vercel, and Raycast — without cloning any single brand.

Key traits:

- near-black background
- gold accent
- terminal-style professional summary
- polished case-study cards
- business value first
- agentic systems positioning
- responsive static GitHub Pages deployment

## Public content safety

The site intentionally uses sanitized case-study summaries and placeholder metrics such as `[X%]`, `[Y]`, and `[Z]` until exact numbers are approved for public sharing.
